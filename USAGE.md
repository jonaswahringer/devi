# Usage

See the `examples` folder for usage samples. Currently, there is a basic example available (`examples/basic.ts`), which demonstrates how to set up and interact with the cache. 

> More examples will be added in the future to cover additional features and integrations.
> E.g., Streaming into NextJS & RSC

## Run with:
```bash
bun --port=8080 --hot examples/server.ts
```
